|图片放大组件

larFancyZoom动态分页组件用于分页查询的结果展示，可以动态调整每页展示结果条数。

|使用方法

$([selector]).larPagination([option]);

|展示效果


|属性说明

属性 类型 描述
pageSize int 每页默认显示的条数
sizeSelector Array 每页显示条数的选择项
key Object
返回结果的属性名定义
                    
    
                      
  1. pageSize : 'pageSize'//每页展示条数的参数名称
    2. 
                      
  2. pageNum : 'curPage'//当前页数的参数名称
    3. 
                      
  3. dataCount : 'allDataCount'//总条数的参数名称
    4. 
                      
  4. pageData : 'curPageData'//当前页数据的参数名称
    5.
remote Object
定义使用服务获取数据的参数，基本与$.ajax的参数一致
                    
    
                      
  1. url : ''//服务路径
    2. 
                      
  2. data : {}//请求时需要发送的参数
    3. 
                      
  3. type : ''//请求方式，一般为POST或GET
    4. 
                      
  4. dataType : ''//返回数据的方式，一般为json
    5. 
                      
  5. succses : Function(data1, data2)//请求成功之后执行的方法，data1为当前页数据数组，data2为请求返回的原始数据
