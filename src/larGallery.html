|轮播图组件

larGallery实现图片轮播的方法

|使用方法

html，只需要一个容器即可，div或section都可以。class最好是写成css-gallery<div id="gallery" class="css-gallery"></div>

js，items是需要轮播图片的数据$('#gallery1').larGallery({  errorImage : 'larGalleryWall/1.jpg',  showNum: 3,  items: [{href:'https://www.baidu.com',src:'larGalleryWall/1.jpg'},          {href:'https://www.baidu.com',src:'larGalleryWall/2.jpg'},          {href:'https://www.baidu.com',src:'larGalleryWall/3.jpg'},          {href:'https://www.baidu.com',src:'larGalleryWall/4.jpg'}]});

css.css-gallery {  width: 1000px;    //该宽度可自由调节  position: relative;  margin: 0 auto;  padding: 0;}.css-gallery-list-wrap {  overflow: hidden;  white-space: nowrap;  position: relative;  margin: 0;  padding: 0;}.css-gallery-list {  position: relative;  padding: 0;}.css-gallery-item {  display: inline-block;  padding: 0 16px;    //可以按需自由调整}.css-gallery-item img {  width: 100%;  height: 300px;    //可以按需自由调整  vertical-align: middle;}.css-gallery-btn.prev {  background-image: url(../../larGallery/left-arrow.png);    //该图标可自行更改  margin-top: -36px;    //可自由调整  position: absolute;  top: 120px;    //可自由调整  left: -40px;    //可自由调整  width: 40px;    //宽高可调整，但必须给值  height: 40px;}.css-gallery-btn.next {  background-image: url(../../larGallery/right-arrow.png);    //该图标可自行更改  margin-top: -36px;    //可自由调整  position: absolute;  top: 120px;    //可自由调整  right: -40px;    //可自由调整  width: 40px;    //宽高可调整，但必须给值  height: 40px;}

|属性说明

  • errorImage
    • 类型：string
    默认值：../../lar-ui/imgs/default.png
    描述：图片加载错误时显示的图片的路径
  • showNum
    • 类型：int
    默认值：3
    描述：当前视图中显示的图片数量
  • items
    • 类型：array
    默认值：undefined
    描述：轮播图中图片的数据。数组中每个对象的属性分别为：href：string，当前图片跳转的路径;src：string，图片路径;extras;object，其中$dom是与当前图片相关的描述的DOM节点的jquery对象，position(string)指的是描述内容相对于图片的显示位置，默认为after图片之后，如果设置为其他值则处理为prepend图片之前
  • speed
    • 类型：int
    默认值：3000
    描述：自动轮播的速度
  • direction
    • 类型：string
    默认值：left
    描述：轮播的方向，left为向左滑动，right为向右滑动，stop为停止滑动
